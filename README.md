Hi, this is my sandbox where I prototype and build random **Rust** crates and libraries for fun. If they turn into something useful I may cut and paste it into it's own repository to maintain it for real, but most crates will just be mock and experimentation code. With that being said, say hello to your friend, **`.unwrap()`**.

The repository is in a mono-repo fashion so I treat each folder in root as it's own repository. Each mono-repo will have it's own ``README.md``, *if I've created one*.
